What is Robotice ?

Robotice software controls large pools of single-board computers, sensors, and other devices throughout a locations, managed through a dashboard or via the Robotice API. Robotice works with popular open source technologies making it ideal for heterogeneous infrastructure.

GENERAL

  • Planning, Monitoring, Reasoning, Reacting
  • Multiplatform - x86 / x64 / armv*
  • Support for more than 10 platforms
  • Distributed but autonomous
  • Web UI, Android, API, CLI
  • 100% open source

PHILOSOPHY

  • Asynchronous by nature
  • High-Performing
  • Slim and lightweight core
  • Integrates with modern technologies
  • Easily extendable

Supported metrics and devices

  • Temperature / Humidity - DHT, TMP36, Hygro Soil
  • Luminosity / Lux / Light - TSL, CDS
  • Turn on / off - relay, Sispm, ..
  • Liquid Flow, Current Sensor

INTERNALS

  • Written in Python
  • Uses Celery
  • Graphite for metrics
  • Openstack Horizon Plugin (dashboard)
  • Robotie like Linux - Ubuntu, Arch

Project Structure

  • Robotice Agent System - Python, Celery, ..
  • Robotice Control System - Django model, API
  • Robotice Dashboard - Horizon(Django) plugin, AngularJS, ..
  • Robotie Mobile Client - hybrid ionic app

Fuzzy

Boiler action

VAR_INPUT
    real_value:       REAL; (* RANGE(0 .. 100) *)
END_VAR

VAR_OUTPUT
    action:    REAL;
END_VAR

FUZZIFY real_value
    TERM Cold := (0, 0) (0, 1) (21, 0) ;
    TERM Good := (0, 0) (21, 1) (21, 0) ;
    TERM Hot := (0, 0) (21, 1) (100, 1) ;
END_FUZZIFY

DEFUZZIFY action
    TERM on := 1 ;
    TERM off := 0 ;
    ACCU:MAX;
    METHOD: Dict;
    DEFAULT := 0;
END_DEFUZZIFY

RULEBLOCK first
    AND:MIN;
    RULE 0: IF (real_value IS Cold) THEN (action IS on);
    RULE 1: IF ((real_value IS Good) OR (real_value IS Hot)) THEN (action IS off);
END_RULEBLOCK

Python code


import fuzzy.storage.fcl.Reader

system = fuzzy.storage.fcl.Reader.Reader().load_from_file("./boiler.fcl")

input = {
    "real_value": 24,
}
output = {
    "action": 0.0
}

system.calculate(input, output)

print output["action"]
